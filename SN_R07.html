Este resultado nos dice que las redes neuronales tienen una especie de universalidad. No importa qué función queramos calcular, sabemos que hay una red neuronal que puede hacer el trabajo.
Esto no significa que una red pueda usarse para calcular exactamente cualquier función. Más bien, podemos obtener una aproximación tan buena como queramos. Aumentando el número de neuronas ocultas podemos mejorar la aproximación. Por ejemplo, antes ilustré una red que calcula alguna función f(x) utilizando tres neuronas ocultas. Para la mayoría de las funciones, solo será posible una aproximación de baja calidad utilizando tres neuronas ocultas. Al aumentar la cantidad de neuronas ocultas (digamos, a cinco), generalmente podemos obtener una mejor aproximación.
